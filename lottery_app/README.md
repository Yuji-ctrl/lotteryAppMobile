# Lottery App - Flutter Frontend Documentation

## Overview
This document provides detailed information regarding the Flutter frontend implementation of the Lottery App.

## Project Setup
1. **Prerequisites**: Make sure you have Flutter SDK installed on your machine.
2. **Clone the Repository**:
   ```bash
   git clone https://github.com/Yuji-ctrl/lotteryAppMobile.git
   cd lotteryAppMobile
   ```
3. **Dependencies**: Run the following command to install dependencies:
   ```bash
   flutter pub get
   ```

## Directory Structure
- `lib/`: Contains the main application code.
- `lib/screens/`: Contains all screen files.
- `lib/widgets/`: Contains reusable widget files.

## Running the App
To run the application, use:
```bash
flutter run
```

## Key Features
- User Authentication: Users can register and login.
- Lottery Ticket Purchase: Users can purchase tickets directly from the app.
- Real-time Updates: Users get real-time updates on lottery results.

## UI Components
### Home Screen
- Shows the available lottery tickets.
- Navigation to other screens.

### Ticket Purchase Screen
- Users can view ticket details and purchase.

### Results Screen
- Displays the latest lottery results.
